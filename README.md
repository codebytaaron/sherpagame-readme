# Sherpa Game

Sherpa Game is a lightweight game built for fun and simplicity.

It is designed to be easy to pick up, quick to play, and focused on straightforward gameplay rather than complex mechanics.

## What this is

- A small game built from scratch  
- Lightweight and easy to run  
- Designed to be intuitive and accessible  
- Focused on enjoyment, not overengineering  

## How to play

- Open the game using the provided link  
- Start playing immediately  
- No setup or installation required  

## Design goals

- Simple rules  
- Fast interaction  
- Clean structure  
- Easy to modify or extend  

## Tech

- Basic game logic  
- Minimal dependencies  
- Clear and readable structure
